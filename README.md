# homebrew-prs-menubar

Homebrew tap for [PRs MenuBar](https://github.com/maiis/prs-menubar) — a macOS menu bar app for viewing pull requests awaiting your review across GitHub, GitLab, and Gitea.

## Install

```bash
brew tap maiis/prs-menubar
brew install --cask prs-menubar
```

## Upgrade

```bash
brew upgrade --cask prs-menubar
```

## Uninstall

```bash
brew uninstall --cask prs-menubar
# or remove all associated data:
brew uninstall --zap --cask prs-menubar
```

## Requirements

- macOS 15 (Sequoia) or later
